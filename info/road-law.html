公道・私道における各種行為の
法的基準とリスク一覧

1. 子供のチョークでの落書き
1. 法的基準（公道）
NG（違法）：軽犯罪法・道路法違反。公道をみだりに汚染する行為として禁止。
2. 暗黙の了解・実態
水で消えるため、住宅街では地域住民に黙認されやすい。
3. 事故・法的リスク
実質的な実害や訴訟に発展するリスクは極めて低い。
4. 私道の場合の見解
所有者（親など）の承諾があれば法的に何ら問題なし。
2. 車を避ける形での子供の遊び
1. 法的基準（公道）
NG（違法）：道路交通法違反。車を避ける前提であっても、公道での球戯や遊戯は一律禁止。
2. 暗黙の了解・実態
住民関係の車しか来ない生活道路では、互助的に黙認されがち。
3. 事故・法的リスク
【高リスク】万が一、車と接触事故が起きた場合、道路で遊んでいた側の過失相殺（過失割合の加算）や、運転手側からの損害賠償請求に発展。
4. 私道の場合の見解
不特定多数が通らない完全な私有地であれば道交法対象外だが、共有私道の場合は住人間でのモラルや合意が必要。
3. 道路の脇等での立ち話
1. 法的基準（公道）
NG（違法）：道路交通法違反。道路脇であっても、他者の通行をわずかでも妨げる目的外の滞留は禁止。
2. 暗黙の了解・実態
「車が来たら道を譲る」という配慮のもと、日常的に広く黙認されている。
3. 事故・法的リスク
立ち話を避けた自転車や歩行者が転倒・事故を起こした場合、原因を作ったとして民事上の損害賠償責任を問われる可能性あり。
4. 私道の場合の見解
通行を完全に塞がない限り、住民（共有持分者）の敷地利用として原則自由。
4. 庭木のはみ出し
1. 法的基準（公道）
NG（違法）：道路法・道路構造令違反。車道上空4.5mの建築限界を1cmでも超えれば一律で違法。
2. 暗黙の了解・実態
「車が通れる範囲がある」と住民が主張し、事実上そのまま放置されやすい。
3. 事故・法的リスク
【高リスク・訴訟問題】はみ出た枝葉を避けようとした車が対向車や歩行者と衝突した場合、樹木の所有者に巨額の損害賠償責任が発生（土地工作物責任）。
4. 私道の場合の見解
私道であっても、建築基準法上の道路（避難経路）であれば通行妨害として行政指導や民事上の剪定請求の対象。
5. 一時的な路上駐車
1. 法的基準（公道）
NG（違法）：道路交通法・車庫法違反。12時間（夜間8時間）未満でも、駐車禁止場所や3.5mの余地がない場合は1分でも違法。
2. 暗黙の了解・実態
来客時や荷降ろしなど、短時間であれば住民間で「お互い様」とされやすい。
3. 事故・法的リスク
【高リスク・即摘発】「近隣からの通報」があれば警察は即座に駐車違反ステッカーを貼る。また、違法駐車車両を避けるための事故が起きた場合、誘発責任として過失を問われる。
4. 私道の場合の見解
一般の人が自由に出入りできない私道であれば車庫法や道交法の対象外。ただし、共有私道で他の住民の通行を妨げれば民事トラブルになる。
6. 常態的な青空駐車（車庫代わりの長時間駐車）
1. 法的基準（公道）
NG（刑事罰対象）：自動車の保管場所の確保等に関する法律（車庫法）第11条違反。公道を特定の車両の保管場所（車庫代わり）として常態的に使用する行為。同じ場所に日中12時間以上、または夜間（日没〜日の出）8時間以上駐車した時点で一発で成立。
2. 暗黙の了解・実態
絶対に認められない（NG）：道路の私物化であり、近隣の多大な迷惑となるため、住民間の「お互い様」や「暗黙の了解」の枠組みから完全に逸脱し、100%トラブルおよび通報の対象となる。
3. 事故・法的リスク
【極めて高いリスク・刑事処分・巨額賠償】近隣住民からの通報（#9110や管轄警察署）により、警察が「事件」として本格的な捜査・取締り・指導を行う。違反者には「3ヶ月以下の懲役または20万円以下の罰金（および違反点数3点）」の刑事罰が科される。また、緊急車両（消防車・救急車）の通行・転回の妨げによる救命遅延や、死角による事故を誘発した場合、刑事上の過失責任のほか、数千万円〜数億円規模の民事上の損害賠償責任を直接負う。
4. 私道の場合の見解
一般の人が自由に出入りできない構造であれば車庫法（警察の取締り）の対象外となる。しかし、分譲地内の共有私道等の場合、特定住民が道路を独占・私物化する行為は他の住民の「通行権の侵害」や「不法占有」にあたり、民事訴訟（土地明渡し請求・損害賠償請求）の対象となる。
💡行政や警察から「指導」があった場合の見解
公道である以上、住民側がどれだけ「車は通れている」「お互い様だ」と反論しても、法律（客観的基準）の前には一切通用しません。
近隣住民から役所や警察に通報が入った場合、行政側は「一律で厳格に法律の条文通り」に指導・排除を行います。
